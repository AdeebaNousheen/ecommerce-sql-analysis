# 🛒 E-Commerce Sales Analysis using SQL

## 📌 Project Overview
Designed and implemented a relational database system to simulate an e-commerce platform and performed business-driven analysis using SQL.

## 🛠 Tools Used
- MySQL (XAMPP)
- phpMyAdmin

## 📊 Key Insights
- Total Revenue: ₹72,000
- Electronics generated highest revenue (₹64,000)
- Headphones were top-selling product
- Identified repeat customer (Aisha Khan)

## 🔍 SQL Concepts Used
- JOINS
- GROUP BY
- HAVING
- Aggregate Functions

- Foreign Keys
